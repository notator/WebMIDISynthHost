This is a simple demo, showing how the ResidentSf2Synth synthesizer from my WebMIDISynthHost project can be used in a Web application.
To use this demo, hover your mouse over the coloured areas below.
Moving the mouse over an area sends the synth a noteOn message.
Moving it out again sends the corresponding noteOff.
